# Terraform Patterns

Terraform modules, state management, workspaces

## What's Included

### Agents
- **Terraform Engineer** - Specialized agent for Terraform modules, state management, workspaces

### Commands
- `/terraform` - Quick-access command for terraform-patterns workflows

### Skills
- **Terraform Patterns** - Pattern library and knowledge base for terraform-patterns

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/terraform analyze

# Use the command with specific input
/terraform generate --context "your project"

# Reference patterns from the skill
"Apply terraform-patterns patterns to this implementation"
```

- Follow established conventions for terraform-patterns
