# Secret Management

Vault, secret rotation, encryption, key management

## What's Included

### Agents
- **Secrets Engineer** - Specialized agent for Vault, secret rotation, encryption, key management

### Commands
- `/secrets` - Quick-access command for secret-management workflows

### Skills
- **Secrets Patterns** - Pattern library and knowledge base for secret-management

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/secrets analyze

# Use the command with specific input
/secrets generate --context "your project"

# Reference patterns from the skill
"Apply secrets-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for secret-management
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
