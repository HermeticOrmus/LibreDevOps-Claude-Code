# Github Actions

GitHub Actions workflows, reusable actions, matrix builds

## What's Included

### Agents
- **Gha Engineer** - Specialized agent for GitHub Actions workflows, reusable actions, matrix builds

### Commands
- `/gha` - Quick-access command for github-actions workflows

### Skills
- **Gha Patterns** - Pattern library and knowledge base for github-actions

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/gha analyze

# Use the command with specific input
/gha generate --context "your project"

# Reference patterns from the skill
"Apply gha-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for github-actions
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
