# Aws Infrastructure

AWS services, CloudFormation, CDK patterns

## What's Included

### Agents
- **Aws Architect** - Specialized agent for AWS services, CloudFormation, CDK patterns

### Commands
- `/aws` - Quick-access command for aws-infrastructure workflows

### Skills
- **Aws Patterns** - Pattern library and knowledge base for aws-infrastructure

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/aws analyze

# Use the command with specific input
/aws generate --context "your project"

# Reference patterns from the skill
"Apply aws-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for aws-infrastructure
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
