# Load Balancing

Load balancer config, traffic management, CDN

## What's Included

### Agents
- **Lb Engineer** - Specialized agent for Load balancer config, traffic management, CDN

### Commands
- `/load-balance` - Quick-access command for load-balancing workflows

### Skills
- **Lb Patterns** - Pattern library and knowledge base for load-balancing

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/load-balance analyze

# Use the command with specific input
/load-balance generate --context "your project"

# Reference patterns from the skill
"Apply lb-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for load-balancing
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
