# Backup Disaster Recovery

Backup strategies, DR planning, RTO/RPO

## What's Included

### Agents
- **Dr Planner** - Specialized agent for Backup strategies, DR planning, RTO/RPO

### Commands
- `/backup-plan` - Quick-access command for backup-disaster-recovery workflows

### Skills
- **Dr Patterns** - Pattern library and knowledge base for backup-disaster-recovery

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/backup-plan analyze

# Use the command with specific input
/backup-plan generate --context "your project"

# Reference patterns from the skill
"Apply dr-patterns patterns to this implementation"
```
