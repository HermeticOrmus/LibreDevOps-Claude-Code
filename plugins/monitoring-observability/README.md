# Monitoring Observability

Prometheus, Grafana, tracing, SLOs/SLIs

## What's Included

### Agents
- **Observability Engineer** - Specialized agent for Prometheus, Grafana, tracing, SLOs/SLIs

### Commands
- `/monitor` - Quick-access command for monitoring-observability workflows

### Skills
- **Observability Patterns** - Pattern library and knowledge base for monitoring-observability

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/monitor analyze

# Use the command with specific input
/monitor generate --context "your project"

# Reference patterns from the skill
"Apply observability-patterns patterns to this implementation"
```
