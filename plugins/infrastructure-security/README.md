# Infrastructure Security

Infrastructure hardening, CIS benchmarks, compliance

## What's Included

### Agents
- **Infrasec Engineer** - Specialized agent for Infrastructure hardening, CIS benchmarks, compliance

### Commands
- `/infrasec` - Quick-access command for infrastructure-security workflows

### Skills
- **Infrasec Patterns** - Pattern library and knowledge base for infrastructure-security

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/infrasec analyze

# Use the command with specific input
/infrasec generate --context "your project"

# Reference patterns from the skill
"Apply infrasec-patterns patterns to this implementation"
```
